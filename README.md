# Evex UI
Evex UI is beautiful web ui component written by react and tailwindcss, unocss. copy and paste for now.

## Features
- High Performance 🚀
- Easy to use 🎉
- Super Beautiful 🔥
- Support All Runtime ❄
- TailwindCSS & UnoCSS 🌈

## Installation

```bash
npx jsr add @evex/ui
bunx --bun jsr add @evex/ui
deno add @evex/ui
```

```ts
import { PoweredBy } from "@evex/ui";

export default function Page() {
    return <>
        ...
        <PoweredBy />
        ...
    </>
}
```
